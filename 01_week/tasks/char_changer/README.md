# Изменение символов

Необходимо написать функцию `CharChanger`, которая изменяет символы по 
следующим правилам:
1. Последовательно повторяющиеся символы, кроме пробела `' '`, заменяются
   одним символом и количеством повторений `SSSS` -> `S4`. Если повторений
   не менее 10, указывается `0` после символа.
1. Все цифры заменяются на `*`, правило 1 действует аналогично, то есть
   `9993` -> `*3*`.
1. Прописные латинские символы не изменяются, правило 1 действует.
1. Строчные латинские символы заменяются на прописные, правило 1 действует
1. Последовательные пробельные символы, заменяются на один без указания числа
   повторений и преобразуются в заданный извне разделитель `delimeter`. 
1. Все остальные символы заменяются на символ `_`

Гарантируется, что последний символ в массиве `'\0'`.

Функция должна возвращать позицию, указывающую за конец строки, где должен
располагаться символ `'\0'`

Пример c разделителем `,`: `1233-fhhhHjo _L3*&((` -> `***2_FH3HJO,_L*___2`

## Примечание

- Разделитель может быть задан символом пробел `' '`, тогда в итоговой строке 
  при наличии нескольких последовательных пробелов останется один.
- Для определения некоторых групп символов есть [специальные функции].
- Задачу можно решить в один проход, выполня замену символов непосредственно
  по месту в массиве.
